Pouvoir coutumier et administratif

Pouvoir coutumier

Le pouvoir coutumier à tiaha est exercé par trois classe d'âge :

  • Les patriarches (Ebebou)
  • Les aspirants (Eberegn)
  • Les soldats (Mab es el)

Les décisions se prennent en Assemblée Générale sur la place publique en présence de ces entités, le Chef de village et les Post-patriarches. Les Post-patriarches sont classés en deux groupes :
  • Les Milakn
  • Les Lel es el

Pouvoir administratif

Le pouvoir administratif est géré par trois entités :

  • La chefferie
  • Le comité de gestion
  • L'association des jeunes
